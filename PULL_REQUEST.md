## HOW TO USE MY WEB APPLICATION

1. Install dependencies:<br>
   `npm i install`<br>

2. Build:<br>
   `npm run build`<br><br>

3. Run the server:<br>
   `npm run preview`<br><br>

4. Open the indicated URL in your web browser, possibly:<br>
   `http://localhost:4173/`
